DRAGONLEVELTIME V1.2.0

This program gives the average time to get a dragon from one level to another,
and the average payout from then exalting the dragon.
It then tells the user how much they could level up a level 1 dragon
in the same amount of time.

USAGE
    python3 dragonLevelTime.py [options] [currentLevel] [desiredLevel]

For a complete list of available options, visit the documentation page:
https://docs.google.com/document/d/1ciYFaxC_Y2XkWomL_RHj6xhsfOYAqd3VGZ1L3u7XdmE/edit

PREREQUISITES:
    For Google Colab use: none
    For use on your own computer: Python 3.0 or higher

INSTALLATION:
    No installation required for Google Colab use

    Install Python 3 here: https://www.python.org/downloads/
